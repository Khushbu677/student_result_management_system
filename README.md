## Student Result Management System

## Description
This is a Python and MySQL based application to manage student records and their results. 
It demonstrates database operations (CRUD), SQL joins, and object-oriented programming concepts.

## Features
- Add, view, update, and delete student information
- Add and view student results with automatic grade calculation
- MySQL database connectivity using Python
- Clean console-based user interface
- OOP design for scalability

## Technologies Used
- **Python** – Programming language
- **MySQL** – Database to store students and results
- **JetBrains DataGrip** – Database management tool

## Installation & Setup
1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/student-result-management-system.git
2. Navigate to the project folder:
   cd student-result-management-system
3. Install required Python package:
   pip install mysql-connector-python
4. Update `db.py` with your MySQL username and password
5. Run the project:
   python main.py
